# 🦷 틀딱 퀴즈

> 추억의 한국 인터넷 문화를 얼마나 기억하고 있나요?  
> 퀴즈를 풀고 나의 **치아 상태**를 확인해보세요.

---

## 소개

버디버디, 무한도전 1기, 아빠와 나... 한 시대를 풍미한 레트로 콘텐츠로 구성된 **추억 퀴즈 앱**입니다.  
정답을 맞힐수록 점수가 쌓이고, 최종 결과는 나의 **치아 건강 상태**로 환산됩니다.

---

## 기능

- **슬라이드 카드 방식** — 이전·현재·다음 카드를 한눈에 보여주는 캐러셀 UI
- **Q / O / X 배지** — 미답변엔 Q, 정답엔 O, 오답엔 X 배지가 카드 모서리에 표시
- **연령대별 정답률** — 문제 풀이 후 10대~40대 정답률 제공
- **최종 결과 페이지** — 누적 점수에 따라 치아 상태 타입 & 메시지 출력
- **반응형 레이아웃** — 모바일 / 데스크탑 모두 대응
